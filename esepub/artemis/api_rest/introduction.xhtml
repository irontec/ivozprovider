Introducción a la API de Ivoz Provider

Ivoz Provider cuenta con tres APIs REST dirigidas por hypermedia [https://en.wikipedia.org/wiki/HATEOAS], una por cada uno de los roles de administración. Todas ellas han sido construidas sobre OpenAPI Specification [https://www.openapis.org/], una especificación abierta liderada por la comunidad dentro de la iniciativa OpenAPI, un proyecto colaborativo de la Linux Foundation. Esta iniciativa cuenta con el respaldo de algunas de las compañias tecnológicas más punteras como Adobe, Google, IBM, Microsoft y muchas otras [https://www.openapis.org/membership/members)].

Se soportan las funcionalidades listadas a continuación:

  • Definición representada de forma legible por personas, sanbox incluido

  • Autorización basada en JSON Web Token (JWT)

  • Negociación de formato de petición y respuesta

  • Paginación de resultados

  • Filtros de propiedades en respuesta

  • Filtro de resultados

  • Orden de resultados

  • Capa de seguridad: Control de acceso a recursos y registros
