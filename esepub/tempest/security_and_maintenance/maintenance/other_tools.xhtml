Otras herramientas

Aunque IvozProvider no incluye ninguna de las herramientas mencionadas aquí, las consideramos cruciales para manejar entornos de producción.

Aquí enumeramos las herramientas configuradas en todas las instalaciones de producción de IvozProvider mantenidas por Irontec [https://www.irontec.com].

Visor de métricas

Interfaz web de Chronograf [https://www.influxdata.com/time-series-platform/chronograf/] mostrando información recopilada por los demás componentes del TICK Stack [https://www.influxdata.com/time-series-platform/], nos permite mostrar:

  • Métricas del sistema en tiempo real.

  • Métricas VoIP en tiempo real.

  • Panel personalizado por perfil.

El componente Kapacitor [https://www.influxdata.com/time-series-platform/kapacitor/] permite configurar alarmas cuando ocurren métricas anómalas o se superan ciertos umbrales.

Monitoreo activo

Todas las instalaciones de IvozProvider mantenidas por Irontec [https://www.irontec.com] son monitoreadas exhaustivamente para resolver problemas lo antes posible y prevenir problemas futuros cuando sea posible.
