El perfil Aux ejecuta software que, aunque no es vital para la realización de llamadas, hace la vida del mantenedor de IvozProvider mucho más fácil.

De hecho, sin ellos, la depuración de problemas sería mucho más difícil y la calidad del servicio prestado se vería afectada.

Aunque IvozProvider no incluye ninguna de las herramientas mencionadas aquí, las consideramos cruciales para tratar con entornos de producción.

Aquí enumeramos las herramientas configuradas en todas las instalaciones de IvozProvider en producción mantenidas por Irontec [https://www.irontec.com]:

  • Captura SIP de Homer: Este increíble software nos permite capturar todo el tráfico SIP para su posterior análisis, para obtener estadísticas, medir la calidad de las llamadas, etc. Visita el sitio web de SIP Capture [http://sipcapture.org/] para más información.

  • Visor de logs de Kibana: Mostrando logs recopilados por los restantes componentes de la pila ELK [https://www.elastic.co/elk-stack].

  • Visor de métricas de Chronograf: Mostrando métricas recopiladas por los restantes componentes de la pila TICK [https://www.influxdata.com/time-series-platform/].
