###########
Other tools
###########

Although IvozProvider does not include any of the tools mentioned here, we consider them crucial for dealing with
production environments.

We list here tools configured in all production IvozProvider installations maintained by
`Irontec <https://www.irontec.com>`_.

**************
Metrics viewer
**************

`Chronograf web interface <https://www.influxdata.com/time-series-platform/chronograf/>`_
showing information collected by remaining `TICK Stack components <https://www.influxdata.com/time-series-platform/>`_,
allow us to show:

- Realtime system metrics.

- Realtime VoIP metrics.

- Custom dashboard per profile.

`Kapacitor component <https://www.influxdata.com/time-series-platform/kapacitor/>`_ allows setting alarms when anomalous
metrics happen or certain thresholds are exceeded.

*****************
Active monitoring
*****************

All IvozProvider installations maintained by `Irontec <https://www.irontec.com>`_ are thoroughly monitored to solve problems
as soon as possible and to prevent future problems when possible.
