Analyzing SIP traffic

Although all production IvozProvider installations maintained by Irontec [https://www.irontec.com] include a Homer SIP Capture Server [https://www.sipcapture.org/], it is not installed in the standalone version of IvozProvider. The reason behind this is that we prefer awesome SIPCAPTURE stack running on an additional machine.

sngrep Ncurses SIP Messages flow viewer developed by Irontec [https://github.com/irontec/sngrep] is currently the preferred tool to inspect SIP traffic included in IvozProvider.

sngrep

See live SIP traffic (all):

sngrep

See live SIP traffic related to calls:

sngrep -c

See live SIP traffic and capture RTP too:

sngrep -c -r

For more reference, visit sngrep official site [https://github.com/irontec/sngrep].

Other capturing tools

Although sngrep is our preferred capturing tool, IvozProvider ships other tools to capture SIP/RTP traffic, such as tcpdump [http://www.tcpdump.org] and ngrep [http://ngrep.sourceforge.net].
